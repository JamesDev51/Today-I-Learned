Call signature

```typescript
type Add = (a: number, b: number) => number;
const add: Add = (a, b) => a + b; //구현과 명세를 분리
```

Overloading

```typescript
type Add = {
  (a: number, b: number): number;
  (a: number, b: string): number;
};

const add: Add = (a, b) => {
  if (typeof b === "string") return a;
  return a + b;
};
```

```typescript
type Config = {
  path: string;
  state: string;
};

type Push = {
  (path: string): void;
  (config: Config): void;
};

const push: Push = (config) => {
  if (typeof config === "string") {
    console.log(config);
  } else {
    console.log(config.path, config.state);
  }
};
```

```typescript
type Add = {
  (a: number, b: number): number;
  (a: number, b: number, c: number): number;
};

const add: Add = (a, b, c?: number) => {
  if (c) return a + b + c;
  return a + b;
};

add(1, 2);
add(1, 2, 3);
```

### Polymorphism (다형성)

다형성을 사용하지 않은 경우 매번 call signature를 추가해줘야 함

```typescript
type SuperPrint = {
  (arr: number[]): void;
  (arr: boolean[]): void;
  (arr: string[]): void;
};

const superPrint: SuperPrint = (arr) => {
  arr.forEach((i) => console.log(i));
};
superPrint([1, 2, 3, 4]);
superPrint([true, false, true]);
superPrint(["a", "b", "c"]);
```

generic을 사용하여 다형 활용

```typescript
type SuperPrint = {
  <T>(arr: T[]): void;
};
const superPrint: SuperPrint = (arr) => {
  arr.forEach((i) => console.log(i));
};
superPrint([1, 2, 3, 4]);
superPrint([true, false, true]);
superPrint(["a", "b", "c"]);
superPrint(1, 2, "a", true); //number | string | boolean
```

```typescript
type SuperPrint = {
  <T>(arr: T[]): T;
};
const superPrint: SuperPrint = (arr) => {
  arr.forEach((i) => console.log(i));
};
const a = superPrint([1, 2, 3, 4]);
const b = superPrint([true, false, true]);
const c = superPrint(["a", "b", "c"]);
const d = superPrint(1, 2, "a", true); //number | string | boolean
```

```typescript
type SuperPrint = <T>(a: T[]) => T;
type SuperPrint = <T, M>(a: T[], b: M) => T;
function superPrint<T>(a: T[]) {
  return a[0];
}
```

```typescript
type Player<E> = {
   name:string
   extraInfo:E
}

const james: Player<{favFood:string}> = {
    name : "james",
    extraInfo: {
        favFood: "chicken"
    }

type JamesPlayer = Player<{favFood:string}>

const james: JamesPlayer = {
    name : "james",
    extraInfo: {
        favFood: "chicken"
    }
}

type JamesExtra = {
    favFood : string
}

type JamesPlayer = Player<JamesExtra>

const nico : Player<null> = {
    name : "nico"
    extraInfo : null
}
```
